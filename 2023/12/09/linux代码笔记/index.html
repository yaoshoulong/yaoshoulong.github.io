Created2023-12-09|Updated2023-12-10

linux代码含义

ls =查看目录下的文件

ls / =访问根目录

隐藏文件：= .xxxxx前有个.

ls -a =可以查看隐藏文件

ls .. =访问上一级文件

touch 文件名 =创建文件/ .文件名 创建隐藏文件

ls -d =只查看目录

ls -al =显示当前目录下所有文件及属性

d:目录/-:文件/l:链接，3个字符一组分为 所有者/所属组/其他人 的权限

r :4/w:2/x:1 读/写/执行

mkdir test=创建文件夹

mkdir -p test1/test2/test3 =创建递归目录

whoami 查看用户

id查看信息

pwd 查看目录

rm -f 文件名=强制删除文件/rm -rf 文件夹=强制删除文件夹/rm -rf /*=删除根目录

ls /bin 查看bin下所有目录，ls /bin | grep(筛选) mv =查找带mv的文件

ls /bin |less查看bin中的可执行文件，q退出

cat xx/xxx查看文本文档/cat -n xxx，查看行号

vi 文件名=编辑文本，i插入模式修改，:wq!保存，:q!退出

touch 文件名 文件名=修改时间一样

chmod 755 xxx=改权限

echo “xxxxx” >> xxx=输出追加

find / -name xx=查找根目录下的关于名字叫xx的文件，-iname =不区分大小写

find / -name ‘test‘ =‘**’相当于查找关键词

useradd admin=创建用户，passwd 用户名=创建密码

ping baidu.com -c 3=linux链接百度测试有没有网

systemctl enable sshd=开机自启动，systemctl start sshd=现在启动

ip add 在虚拟机cmd中输 入 ssh root@192.168.xx.xx链接

用户管理代码

添加用户

命令格式格式：useradd [命令参数] admin

删除用户

命令格式：userdel [命令参数] admin

密码管理

命令格式：passwd [命令参数] 参数

修改密码:

命令格式：chpasswd [命令参数] 参数

用户登录:

login [命令参数] 参数

退出:logout或者exit

切换身份:

命令格式：su [命令参数] 参数

用户高级管理

创建用户组:

命令格式：groupadd [命令参数] groupID

删除用户组:

命令格式：groupdel 参数

查看用所属组

命令格式：id [命令参数] 参数

命令格式：groups 参数

修改所属组

命如果要一个用户仅仅属于某一个组，而不是同时属于多个组，则使用如下命令：

命令格式：usermod -g 组名 用户名

如果要给一个用户添加一个新的组，同时保留原来有所的组，则使用如下命令：

命令格式：usermod -a -G 组名 用户名

使用useradd命令创建用户的时候指定用户组

命令格式：useradd -g 组名 用户名

命令格式：useradd -G 组名 用户名

查看用户属性

命令格式：finger [命令参数] 参数

用户信息修改

命令格式：usermod [命令参数] 参数

用户组信息修改

命令格式：groupmod [命令参数] 参数

创建文件

具体命令如下：rm 参数 文件名

删除两个文件：rm 文件名 文件名

具体命令如下：mkdir 参数 目录名

删除目录

具体命令如下：rmdir 参数 目录名

具体命令如下：cp 参数 源文件 目的目录

cp 源文件 目的目录/重命名

重命名文件

具体命令如下：mv 参数 目录名（类似于移动和剪切）

cp -r Dir1 Dir2 Dir3

重命名目录

具体命令如下：mv 参数 源目录 目的目录

查看文件内容

具体命令如下：cat 参数 文件名

具体命令如下：head 参数 文件名

具体命令如下：tail 参数 文件名

查看目录内容

具体命令如下：ls 参数 目录

修改文件权限命令

文字设定法具体命令如下：

chmod ［who］ ［+ | - | =］ ［mode] 文件名

数字设定法具体命令如下：

chmod ［mode］ 文件名

mode为具体的数字

修改所有者权限

具体命令如下：

chmod u ［+ | - | =］ ［mode] 文件名

修改其他用户权限

Linux修改其他组用户权限的方法与修改所有者的权限大致一样，唯一不同的点是修改其他组用户权限采用chmod o而不是采用chmod u。

修改文件所有者命令

具体命令如下：

chown 用户名/用户ID 文件名

修改目录所有者命令

具体命令如下：

chown 参数 用户名/用户ID 目录名

修改文件所属组命令

具体命令如下：

chgrp 用户组名/组ID 文件名

具体命令如下：

chown:用户组名/组ID 文件名

如果用户想使用chown命令同时修改所有者和所属组信息，则可以使用如下：

chown 用户名/用户名ID:用户组名/组ID 文件名

修改目录所属组命令

具体命令如下：

chgrp 参数 用户组名/组ID 目录名

利用du命令查看文件大小

具体命令如下：

df 命令参数 参数

硬链接

具体命令如下：

ln 源文件 目标文件

软链接

具体命令如下：

ln -s 源文件 目标文件

setuid，setgid

chmod u+，-权限 文件/目录

chmod g+，-权限 文件/目录

stick bit

chmod o [+|-] t 目录

修改文件/目录特殊属性

具体命令如下：

chattr [+|-|=] 属性标示符 命令参数 文件/目录

查看文件/目录特殊属性

具体命令如下：

lsattr 命令参数 文件/目录

打包

具体打包命令如下：

tar -cvf 生成的tar包名 要打包的文件/目录列表

解包

具体打包命令如下：

tar -xvf 解压包名 -C 解压到指定目录
