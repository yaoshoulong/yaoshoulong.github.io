linux代码含义

ls =查看目录下的文件

ls / =访问根目录

隐藏文件：= .xxxxx前有个.

ls -a =可以查看隐藏文件

ls .. =访问上一级文件

touch 文件名 =创建文件/ .文件名 创建隐藏文件

ls -d =只查看目录

ls -al =显示当前目录下所有文件及属性

d:目录/-:文件/l:链接，3个字符一组分为 所有者/所属组/其他人 的权限

r :4/w:2/x:1 读/写/执行

mkdir test=创建文件夹

mkdir -p test1/test2/test3 =创建递归目录

whoami 查看用户

id查看信息

pwd 查看目录

rm -f 文件名=强制删除文件/rm -rf 文件夹=强制删除文件夹/rm -rf /*=删除根目录

ls /bin 查看bin下所有目录，ls /bin | grep(筛选) mv =查找带mv的文件

ls /bin |less查看bin中的可执行文件，q退出

cat xx/xxx查看文本文档/cat -n xxx，查看行号

vi 文件名=编辑文本，i插入模式修改，:wq!保存，:q!退出

touch 文件名 文件名=修改时间一样

chmod 755 xxx=改权限

echo “xxxxx” >> xxx=输出追加

find / -name xx=查找根目录下的关于名字叫xx的文件，-iname =不区分大小写

find / -name ‘test‘ =‘**’相当于查找关键词

useradd admin=创建用户，passwd 用户名=创建密码

ping baidu.com -c 3=linux链接百度测试有没有网

systemctl enable sshd=开机自启动，systemctl start sshd=现在启动

ip add 在虚拟机cmd中输 入 ssh root@192.168.xx.xx链接
