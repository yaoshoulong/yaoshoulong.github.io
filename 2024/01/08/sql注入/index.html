python盲注，延时注入脚本

  1. 盲注的布尔型（Boolean-based Blind SQL Injection）
    在布尔型盲注中，攻击者通过观察应用程序的响应，来推断条件语句是否为真。攻击者通常使用布尔逻辑运算，如AND和OR，以及条件语句的真假情况来判断数据库中的数据。
  2. 盲注的时间型（Time-based Blind SQL Injection）
    在时间型盲注中，攻击者通过在SQL语句中引入时间函数，观察应用程序响应的延迟来推断条件语句是否为真。
